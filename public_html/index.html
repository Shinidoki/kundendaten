Kundendaten



  • Klicken Sie auf den "Datei hochladen"-Button und wählen sie die exportierte Datei im ".CSV"-Format aus.
  • (Nun sollten die Daten in der Tabelle angezeigt werden.)
  • Mithilfe des "Spalten"-Buttons unten Links an der Tabelle können Sie auswählen welche Spalten angezeigt werden sollen.
  • Die Daten können gefiltert werden, indem in den Feldern unter dem Spaltennamen ein Suchbegriff eingegeben wird.
  • Mit einem Klick auf den Spaltennamen kann nach dieser Spalte sortiert werden (auf-/absteigend)
